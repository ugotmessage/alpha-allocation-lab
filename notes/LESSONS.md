# LESSONS

這裡記錄研究過程中值得保留的經驗。

## 用途
- 避免未來重複犯同樣錯誤
- 累積哪些修正方向通常有效
- 讓未來的 OC / Agent 有可讀的長期研究記憶

## 建議格式
- Date
- Related hypothesis / experiment
- Observation
- Lesson
- Recommended next action
