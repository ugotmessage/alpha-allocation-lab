# 可用資料規格

## 價格資料
- open
- high
- low
- close
- volume

## 可用技術指標
- MA
- EMA
- RSI
- ATR
- MACD
- rolling high / rolling low

## 若有可用則列入
- 基本面資料
- 籌碼資料
- 產業分類
- 市值
- 波動度
- 大盤環境指標

## 嚴格限制
- 不得使用未來資料
- 不得使用無法穩定取得的欄位
- 不得使用人工臨時補充但無法重現的資料
- 若 XS Code 使用平台特定內建函數，必須先標示是否可在外部回測重現

## 待補充欄位
請在此補上你實際擁有的資料表、欄位名稱、頻率（日K / 週K / 分K）與資料來源。
