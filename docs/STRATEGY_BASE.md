# 主策略骨架

## 策略目的
針對已篩選出的優質股票，尋找較佳波段進出點，吃到主要漲幅，避免無效持有。

## 股票池
- 僅限已通過基本面 / 產業 / 籌碼篩選的優質股票
- 本策略不負責選股，只負責 timing 與配置

## 核心概念
- 只做強勢中的整理後再發動
- 避免追過度延伸
- 優先尋找可延續波段的買點
- 賣出以保住波段利潤為主，不強求最高點

- 在此補上你的原始策略進場邏輯
- 可直接用條列或貼上 XS Code 後再由 Agent 協助拆解

## 基本出場邏輯
- 在此補上你的原始策略出場邏輯
- 可包含停損、停利、移動停利、先減碼等概念

## 加減碼概念
- 初始倉位不是滿倉
- 若突破後確認，可加碼
- 若不如預期，應快速減碼或撤退

## 不做的情況
- 爆量長黑後立即接
- 大盤轉弱時逆勢重壓
- 籌碼結構不明時進場
