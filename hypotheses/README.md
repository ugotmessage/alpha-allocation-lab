# Hypotheses

這個資料夾用來存放每一個研究假設（hypothesis）。

## 命名建議
- `hyp_YYYYMMDD_slug.md`

## 每份 hypothesis 建議包含
- title
- thesis
- expected edge
- market / universe
- timeframe
- expected regime
- risks
- validation plan
- related idea id

## 原則
- 一份 hypothesis 只表達一個核心想法
- 不要把多個完全不同的策略混在同一份
- 每個 hypothesis 都應該能對應至少一份 experiment
