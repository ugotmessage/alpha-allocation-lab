# Experiments

這個資料夾用來存放每一輪研究實驗結果。

## 命名建議
- `exp_YYYYMMDD_slug.md`

## 每份 experiment 建議包含
- baseline version
- candidate version
- changed variables
- dataset / time range
- evaluation metrics
- failure analysis
- promotion decision
- next action

## 原則
- 不只記錄成功案例，也要記錄失敗案例
- 盡量保留完整脈絡，讓未來的 OC / Agent 可以接手
- 每份 experiment 應可回溯到對應 hypothesis
